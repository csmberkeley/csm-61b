\part Consider a list of N Strings. Assuming that the max length of a String is N, what is the runtime of inserting all the strings into a hashmap? Assume that Java does not precompute the hashcodes of Strings.
\begin{solution}[0.75in]
$O(N^2)$
\end{solution}
